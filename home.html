AR that works on Android and iOS Devices!

Note : GLB and USDZ links have to be served through https protocol

Include the Charpstar AR Script

The script is very small (~8KB Size) and tells the browser to open the AR links in their respective default applications and additionally does AR Support Checks

<script src="https://cdn.jsdelivr.net/gh/charpstar/InPlace/charpstar-viewer.js"></script>

Include the AR Code inside an AR Badge Image or Button element

  • The Image (Which can be anything like an AR Icon/Badge) or Button acts as the activation element for the AR experience. You can style this however you want

  • We hide the element (Not in this demo) on unsupported devices and browsers with our script by selecting its class property "charpstarARViewer"

  • The 'data-androidSrc' and 'data-iosSrc' attributes should contain the links to the GLB and USDZ links that we will provide you

  • The 'data-desktopQR' attribute enables a direct QR Link to the AR model and makes the AR Button visible on desktops. Remove this if you don't need this feature

<img class="charpstarARViewer" onclick="activateAR(this)" src="https://charpstar.se/AR.svg" data-desktopqr data-androidsrc="https://charpstar.se/InPlace/Demos/ClasOhlson/ClasOhlson.glb" data-iossrc="https://charpstar.se/InPlace/Demos/ClasOhlson/ClasOhlson.usdz">
